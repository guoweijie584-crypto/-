# Moving World Background

## Goal

Make the canvas background visibly change as the player moves, so movement feels like traveling through the ancient city rather than moving over a static wallpaper.

## Plan

- Keep the distant sky/moon/silhouette as a screen-space atmosphere layer.
- Render a world-space ground layer inside the camera transform, including city bounds, map features, path/grid texture, decorations, and landmark labels.
- Add focused render coverage that checks the world layer uses camera-relative visible bounds.

## Verification

- Run the focused rendering test.
- Run the full Vitest suite if the focused test passes.
